添加 SSH-KEY 信息到 GitHub或GitLab

密钥位置在：C:\Users\你的用户名\.ssh 目录下，找到 id_rsa.pub 并使用编辑器打开

登陆GitHub，点击头像 -> Settings -> SSH and GPG keys -> New SSH key，然后复制上面的公钥内容粘贴进“Key”文本域内，“Title”文本域随便起个名字，点击 Add key。
